## 📄 Soal yang harus dikerjakan :
- Melakukan git push menggunakan git bash

dengan perintah :
```
git init // untuk melakukan inisialisasi repository baru
git remote // agar masuk kedalam repository yang akan dikerjakan
git add “namaspesifikfile.html” ( git add . (untuk memilih semua file))// untuk menambahkan file secara spesifik kedalam staging area sebelum akhirnya di upload ke dalam repository
git commit -m “pesan” // untuk memberi pesan perubahan pada file yang akan di upload ke repository nantinya
git push // untuk melakukan push file atau mulai mengupload file yang sudah di add dengan perintah git add “namafile.html” ke dalam repository yang di remote
git push -u origin main // ini sama dengan git push yang sebelumnya. cuma, disini sekaligus merubah filenya akan di simpan ke branch mana
git pull // untuk melakukan import file yang sudah ada pada repository ke local atau komputer kita
```
